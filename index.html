最近单位换了银河麒麟的系统，电脑是飞腾CPU，还带个身份鉴别的UKEY，这怎么能忍 1.银河麒麟官网→下载飞腾版银河桌面系统→制作U盘启动盘→安装2.启动后，查看/etc/apt/sources.list,然后sudo chattr +i /etc/apt/sources.list3.dpkg -r –force-remove-reinstreq cmake cmake-data4.sudo apt-get update→sudo apt-get update→sudo apt-get install -f→dpkg –configure -a→重启5.打开软件商店，点开移动应用，下载，等待安装完成，重启6.sudo kylin-system-verify，按要求一步步操作7.搞定 然后在windows电脑装个filezilla server，被动模式设置上内网IP地址，设置上用户和根目录，麒麟这边的IP地址、子网掩码、网关跟Windows电脑设置在一张网上，一根网线连两台电脑，开传！

近况报告 自从去年9月份考得一份财政的事业编以后，我的工作被很多人羡慕（邻居、父母的同学和朋友们，等等）。但是半年多过去了，我却觉得这是一个围城。原本以为我逃离了工厂的苦海，没想到又跳进了另一个深坑😂尤其是近一段时间，我总是觉得自己非常苦闷。这份感觉主要来自一下几个方面：（一）在别人看来很简单的事情，到了我的手里却经常搞砸（二）没办法融入到圈子。我和同事们基本没有什么共同语言。这就导致我不知道和同事们都聊些什么。话不多，人看起来还闷闷不乐，其他人愿意跟你交流那就怪了。（三）亲人们的不理解。事业编虽然比公务员差一些，但好在稳定。所以别人都很羡慕，说这份“工作”很好，有个班上很好。可我并没有觉得我的工作好在哪里。我的生活完全被工作所占据着。工作让我心焦，让我没有更多精力去想其他的事情。这我仿佛又想起我在工厂的那段时间。（四）对自己的厌恶。我讨厌自己的形象，我讨厌自己的性格，我认为自己不如其他人。每当其他人对我夸奖，我都会怀疑他们说的话是不是在讽刺我。我经常怀疑我自己是不是抑郁症。我甚至有一段时间觉得我是不是疯了，觉得我的脑子是不是有什么问题
